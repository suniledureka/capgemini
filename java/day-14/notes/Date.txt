y   = year   (yy or yyyy)
M   = month  (MM)
d   = day in month (dd)

z   = time zone  text        (e.g. Pacific Standard Time...)

Comparing Two Dates

Java program to compare two Date instances using the compareTo() method.

Date date1 = new Date();
Date date2 = new Date();

int comparison = date1.compareTo(date2);

The compareTo() method returns:

    An int larger than 0 if the date the method is called on is later than the date given as parameter.
    An int value of 0 if the dates are equal.
    An int value less than 0 if the date the method is called on is earlier than the date given as parameter.

DateTimeFormatter df = DateTimeFormatter.ofPattern("sd-MMM-yyyy");
LocalDate  d1 = LocalDate.parse(date1, df);
LocalDate  d2 = LocalDate.parse(date2, df);

Long datediff = ChronoUnit.DAYS.between(d1,d2);
